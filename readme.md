# WebC component library

## Setup

Install dependencies.

```sh
yarn install
```

Run Eleventy to render WebC components to HTML Storybook can understand.

```sh
yarn start
```

Keep Eleventy running.

Then in a new terminal Run Storybook.

```sh
yarn storybook
```
